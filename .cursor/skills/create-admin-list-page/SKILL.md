# Skill: Create Admin List Page

## 使用场景

创建任何以"资源列表"为主体的管理页面，包括但不限于：
租户列表、项目列表、Agent 列表、Skills 列表、终端列表、用户列表、模板列表、凭证列表。

---

## 必须完整实现的能力清单（逐项确认，不得遗漏）

### 1. 顶部工具栏（使用 ListPageToolbar 组件）
- [ ] 新建按钮（主操作，触发新建抽屉或跳转）
- [ ] 关键词搜索框
- [ ] 状态筛选下拉
- [ ] 类型/分类筛选（如资源有多类型）
- [ ] 批量操作区（选中行数 > 0 时显示"已选 N 项"+ 批量删除按钮）

### 2. 表格列（使用 Table 组件）
- [ ] 第一列：Checkbox（支持全选/单选）
- [ ] 名称列：可点击，跳转详情页或打开详情抽屉
- [ ] 状态列：使用 StatusTag 组件，显示中文枚举标签
- [ ] 关键业务字段列（视资源类型）
- [ ] 时间列（创建时间或更新时间，至少一列）
- [ ] 操作列（固定最右侧）：查看、编辑、删除

### 3. 行级操作
- [ ] **查看**：跳转详情页或打开只读抽屉
- [ ] **编辑**：打开编辑抽屉，预填当前行数据
- [ ] **删除**：打开二次确认 Dialog，显示被删除对象名称，确认后调用 service 删除，成功后刷新列表
- [ ] **状态切换**（如资源有启用/禁用）：切换后刷新行状态

### 4. 删除确认弹窗（使用 Dialog 组件）
- [ ] 标题：「确认删除」
- [ ] 正文：「确定要删除「{name}」吗？此操作不可撤销。」
- [ ] 取消按钮 + 确认删除按钮（危险色）
- [ ] 删除成功：提示"已删除"，刷新列表
- [ ] 删除失败：显示错误原因，弹窗关闭，列表不变

### 5. 批量删除
- [ ] 选中行后，工具栏显示"已选 N 项 [批量删除]"
- [ ] 触发确认弹窗："将删除 N 个 {资源名}，不可撤销"
- [ ] 完成后清空选中，刷新列表

### 6. 空状态（使用 EmptyState 组件）
- [ ] 无数据时显示 EmptyState，说明为空原因，提供新建按钮
- [ ] 搜索/筛选后无结果时，显示"未找到匹配项"，不显示新建按钮

### 7. 分页（使用 Pagination 组件）
- [ ] 显示总条数
- [ ] 支持切换每页条数（10/20/50）
- [ ] 翻页保留搜索/筛选条件

### 8. 加载与错误态
- [ ] 数据加载中：表格显示 loading 状态
- [ ] 加载失败：显示错误提示 + 重试入口

---

## 分层要求

```
页面（page）：只做布局、状态绑定、调用 service
  ↓
service：list / create / update / delete / patchStatus
  ↓
repository：mock 数据读写
  ↓
schema：资源类型定义、状态枚举
```

页面不得直接操作 repository 或写业务逻辑。

---

## 禁止行为

- 禁止跳过删除确认弹窗直接删除
- 禁止在页面组件内写 mock 数据
- 禁止重新创建已有的 Table / StatusTag / Dialog / EmptyState 等通用组件
- 禁止行操作超过 4 项时不用下拉收纳
